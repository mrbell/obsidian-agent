# 9 — Readwise Integration

**Status**: `completed`

## Description

Pull highlights, notes, and article metadata from the Readwise API into the Obsidian
vault. For each article with highlights, create a permanent note in a dedicated
`Readwise/` folder. Notes land there via the existing promoter pipeline rather than
through BotInbox, since they are permanent reference notes rather than inbox items.
