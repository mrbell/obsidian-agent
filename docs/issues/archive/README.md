# Archived Issues

This folder contains completed issue trees that are kept for historical and design context
but removed from the active `docs/issues/` working set to reduce noise.

Archiving rule:

- If a parent issue is completed and all of its children are completed, move the parent and
  all of its children here together.
- If a parent issue is still open or in progress, keep its completed children in the main
  `docs/issues/` folder.
